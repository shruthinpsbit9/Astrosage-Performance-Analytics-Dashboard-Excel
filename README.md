# Astrosage Performance Analytics Dashboard

## 📊 Overview
Developed an interactive Excel dashboard analyzing 28K+ users and 149 gurus to uncover service performance trends and user behavior patterns.

## 🔍 Key Insights
- 26% service completion rate  
- 56.63% retention rate  
- 65% low ratings (average rating: 2.9)

## ⚠️ Operational Gaps
- 30.37% uncategorized chats  
- 41.99% uncategorized calls  
- 49.76% incomplete chats  
- 19.15% unanswered calls  

## 💡 Recommendations
- Improve chat and call categorization  
- Reduce incomplete interactions  
- Enhance service quality and user experience
